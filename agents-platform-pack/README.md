# 🤖 Plataforma de Agentes IA — CRM AI-First

Funcionários digitais dentro do seu CRM: agentes de IA que **agem** —
consultam seus números, atendem leads no WhatsApp, criam tarefas, cobram
o time no grupo e trabalham sozinhos em rotinas (ex: relatório toda manhã).

**Vem com 5 agentes prontos:** 🧭 CEO · ✍️ Conteúdo · 📊 Tráfego ·
🧑‍💼 Gestor de Time · 💆 Atendente

## Como instalar

1. Descompacte este zip na **raiz do projeto do CRM**
2. Abra o Claude Code no projeto e diga:

> **"instala a plataforma de agentes"**

O Claude segue o `INSTALL.md` e faz tudo: banco, funções, frontend e
templates. Em ~15 minutos você cria seu primeiro agente.

Requisito: CRM AI-First já instalado e funcionando.
