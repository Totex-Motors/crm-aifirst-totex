## Como configurar (3 passos)

### 1️⃣ Crie essas contas ANTES (grátis)

Vai precisar dessas chaves. Cria agora, guarda num bloco de notas:

| Serviço | Pra que serve | Link | Custo |
|---|---|---|---|
| **Supabase** | Banco de dados | https://supabase.com | Grátis |
| **Anthropic** | IA do agente (Claude) | https://console.anthropic.com | $5 grátis |
| **Google Gemini** | Transcrição de áudio/imagem | https://aistudio.google.com/apikey | Grátis |
| **UAZAPI** | Conectar WhatsApp | https://uazapi.com | ~R$30/mês |

### 2️⃣ Instale o Claude Code

1. Abre https://claude.com/claude-code
2. Baixa e instala
3. Reinicia o computador

### 3️⃣ Abra o projeto no Claude Code

1. Extrai o ZIP num lugar fácil (ex: Desktop)
2. Abre o **Terminal**:
   - Mac: `Cmd + Espaço` → digita "terminal"
   - Windows: tecla Windows → digita "cmd"
3. Entra na pasta do projeto:
   ```bash
   cd ~/Desktop/crm-template
   ```
4. Abre o Claude Code:
   ```bash
   claude
   ```
5. Quando abrir, **digita**:
   ```
   oi
   ```
   E aperta Enter.

**Pronto.** O Claude assume daqui e te guia até o agente responder no seu WhatsApp.
